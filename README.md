# douki

研究用プログラム

## 依存ライブラリ

以下のライブラリがインストールされているか確認してください。(brewでインストール)

- portaudio 
- fftw

## インストール

```shell script
make
sudo make install
```

## アンインストール

```shell script
make uninstall
```
